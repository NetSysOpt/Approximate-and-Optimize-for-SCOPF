# Approximate-and-Optimize for SCOPF

Security-Constrained AC Optimal Power Flow`

Julia:
-	InfrastructureModels 0.7.8
-	Ipopt 1.6.0
-	JSON 0.21.4
-	JuMP 1.18.1
